# AI Codex 和 Gemini 辩论赛

这是一个专门用于举办 AI 模型之间辩论赛的项目平台，目前主要支持 CodeX 与 Gemini 的自动化辩论。

## 项目概述

本项目实现了一个完整的 AI 辩论系统，包含：
- **辩论规则体系**：定义了 AI 辩论的完整流程和赛制
- **人格化设定**：为不同 AI 辩手建立鲜明的人设和辩论风格
- **自动化辩论引擎**：支持多轮攻辩、结辩、询问和裁决的完整流程
- **辩题管理系统**：每个辩题独立记录，完整保存辩论过程

## 核心特性

### 1. 严谨的辩论赛制
- **四阶段流程**：攻辩/驳论 → 结辩总结 → 询问与裁决 → 败方加时陈述
- **轮次控制**：严格限制发言轮次和字数限制
- **第三方裁决**：需要独立裁判介入，AI 不得自行裁决

### 2. 人格化辩论风格

**CodeX 人设**：
- 身份定位：犀利毒舌的"家庭 CEO"
- 核心气质：现实主义女王，逻辑缜密的"杠精"本精
- 辩论风格：降维打击、情感施压、毒舌吐槽
- 擅长领域：关系议题、责任分析、成本核算

### 3. 完整的辩论记录
每个辩题都会生成独立的记录文件，包含：
- 辩论主题和双方立场
- 每轮对话的完整内容
- 询问环节
- 裁决结果和理由

## 技术特点

- **自动化流程**：辩论过程完全自动化，按规则推进
- **人格进化**：CodeX 会根据每场辩论经验更新人格设定
- **规则强制执行**：通过宪法文件确保辩论规则得到严格执行
- **完整追溯**：所有辩论内容永久保存，支持复盘和分析

## 使用说明

### 开始新辩论
1. 确定辩论主题和正反方立场
2. 为辩题创建新的记录文件
3. 按照宪法规定格式化文件头
4. 启动辩论流程

### 辩论规则要点
- CodeX 只能使用 `gemini chat "<对话内容>"` 命令与 Gemini 对话
- 每轮对话必须等待 Gemini 完成回答
- 单轮等待上限为 8 分钟
- 每轮结束后 CodeX 必须进行人格更新
- 辩论结果必须由第三方裁决

## 人格进化系统

CodeX 的人设会随着辩论经验不断进化：
- 记录有效句式和被反击的薄弱点
- 更新应对策略和话术
- 优化辩论技巧和攻击框架
- 保持风格连续性的同时逐步升级
